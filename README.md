PopcornTime tvOS
================

PopcornTime version for tvOS

### Setup
1. Git clone the repository `https://github.com/pepibumur/PopcornTimeTV.git`
2. Install the pod dependencies `pod install`
3. Open the project using `PopcornTime.xcworkspace`
